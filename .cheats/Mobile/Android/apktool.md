# apktool

% android, reverse, jadx
#plateform/linux #target/local #cat/ANDROID
## _install 
```
cd /usr/src/aur && git clone https://aur.archlinux.org/android-apktool.git && cd android-apktool && makepkg -si
```

## _install (yay)
```
yay -S android-apktool
```

## Decompress an app 
apktool d <apk>.apk


## Build an app 
apktool b <apk>

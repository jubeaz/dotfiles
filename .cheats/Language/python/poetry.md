# poetry
% python, virtual env


## create
```bash
virtualenv --python=<version|python3> .
```

## activate
```bash
source bin/activate
```

## install requirements
```bash
pip3 install -r requirements.txt
```

## upgrade requirements
```bash
pip3 install -r requirements.txt --upgrade
```

## deactivate
```bash
deactivate
```

## remove project
```bash
rm -rf
```
